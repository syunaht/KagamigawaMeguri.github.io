BurpSuite Pro 2021最新版破解激活教程及下载

更新日期:   2021-11-21

前言

还有很多师傅不知道，portswigger其实已经公开了各个版本的下载地址了。配合之前的keygen loader，就可以一直保持最新版破解。

正好要给新生做培训，一个一个手把手弄太累了，所以写篇教程方便授课。

下载

官方jar包

https://portswigger.net/burp/releases

下载pro的jar包就行。

为方便后续操作，请下载完成后将文件名改为burpsuite_pro.jar

旧版+新版loader

点击蓝奏云下载，密码:18yk

激活

windows/linux通用

注意：在有些java版本，比如单纯的jre1.8是可以点击jar包启动而不用输入命令启动，但现在新版的burpsuite都推荐jdk11，国外论坛甚至推荐jdk14。故点击启动不在讨论范围内。

1.启动loader

打开命令行，输入：

java -Dfile.encoding=utf-8  -noverify -javaagent:BurpSuiteLoader.jar -jar burpsuite_pro.jar

一行代码的事就不需要写脚本了吧。

点击Next

点击Manual activation

然后会出现一大堆码，复制一下。

2.启动旧版loader。

java -jar burploader-old.jar

然后将刚刚复制的一大串代码粘贴到Actvation Request里

之后下方的Activation Response会生成激活码，复制粘贴到之前那个框框就行了。

以后启动都需要用loader启动，且更新后也不需要再次激活。

一键启动

一般启动burpsuite都是用环境变量里的java，但这种方式不够自由！

所以在此祭出我的独家启动秘方！

CreateObject("WScript.Shell").Run """C:\Program Files\Java\jdk-14.0.2\bin\java.exe"" -Dfile.encoding=utf-8  -noverify -javaagent:BurpSuiteLoader.jar -jar burpsuite_pro.jar""",0

代码复制下来，把java路径改一下，文件改为vbs后缀

linux的话就自个写sh吧。
